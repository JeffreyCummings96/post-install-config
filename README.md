 To configure <b> Agents (workers)</b> do the following Admin Panel -> Agents -> Add New

 <img src=https://i.imgur.com/sYDmj4Y.png/>
</p>
<p>
 To configure <b>Users (Customers)</b> do the following Agent Panel -> Users -> Add New
